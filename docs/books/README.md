# Vapor Books

Status: mdBook stub index.

This folder holds the planned mdBook set for Vapor.

- `index`: routing book for the full docs set.
- `vapor`: the main concept and product book.
- `sdk`: the SDK user book.
- `launcher`: the Launcher user book.
- `distribution`: Steam, Workshop, Registry, trust, and release book.
- `internals`: contributor and architecture book.
- `cookbook`: recipe book.

Each book uses the normal mdBook shape:

- `book.toml`
- `src/SUMMARY.md`
- `src/*.md`
