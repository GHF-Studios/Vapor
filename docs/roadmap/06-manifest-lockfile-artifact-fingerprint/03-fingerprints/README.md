# Fingerprints

Status: command surface only.

## Exists Now

- SDK and Launcher command models include fingerprint read commands.
- No fingerprint implementation exists.

## Needed Next

- Decide the first fingerprint input.
- Decide whether the first form is semantic text/TOML, hash, or both.
- Decide where fingerprints are stored before registry enforcement exists.

## Total Target

Fingerprints let tools compare content identity, compatibility, and integrity without pretending to solve hostile-code security.
