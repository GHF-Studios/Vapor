Decision Ledger 020d-GM-a: Glossary Migration Branch Queue

Status:
This is a nested branch under decision ledger 020.
It pauses the main 020 trunk at Pivot Check 1 in `020d_referee_queue.txt`.
It exists to resolve and execute the canonical glossary migration workload before the main queue resumes as `020e`.

Branch rule:
Do not continue Active Repo Topology or later 020 trunk sections until this branch converges or the owner explicitly aborts/deprioritizes it.

Branch output target:
Produce an owner-approved glossary migration packet with exact source paths, destination paths, selected pages, excluded files, status policy, dangling-link policy, and no-rewrite guarantees. Only after that packet is approved should actual glossary files be copied into `Vapor/docs`.

Carried decisions:

- `Loo-Cast/loo_cast_alpha/docs/glossary` is the source corpus for canonical Vapor concept material.
- The old `Loo-Cast` repo remains legacy/context workspace.
- "Legacy glossary" means legacy physical location/history, not non-canonical concept content.
- `Vapor/docs/glossary/` is accepted.
- A parallel `Vapor/docs/tech_glossary/` is desired, but the old split must be scrutinized.
- Migration is selective. Spacetime and Loo Cast are not the current focus.
- Old AI logs stay history.
- Old diagrams, RFCs, and root docs stay out unless separately approved.
- Preserve Obsidian-style wikilinks, frontmatter, and tags initially.
- Exclude/nuke `.obsidian`; nothing significant there matters.
- Linked but not-yet-migrated pages may remain dangling until intentionally introduced.
- Imported pages default to `legacy signal` until individually approved as `WIP` or `implemented`.
- `implemented` is reserved for concepts directly represented in active code or repo-local config.
- `WIP` means canonical but not trustworthy/final.
- `legacy signal` is one-way transition status for old imported pages.
- Summary docs and the alignment ledger may live in `Vapor/docs/` root, but only with restraint.
- Even a tiny `Vapor/docs/README.md` requires a drafted approval packet first.
